## 魔兽钓鱼脚本

+ 施放钓鱼
+ 鱼漂识别
	+ 抓取屏幕图片
	+ 对指定区域的图片内容进行（不需要全屏，以加快运算速度）
	+ win11 设置屏幕进行了缩放，需要进行坐标比例换算
	+ 对屏幕中的点，进行颜色弱化，只获取RGB值最高的值，其余的值作为0处理，筛选出指定颜色（如红色, 需要根据游戏中环境颜色进行调整）的点
	+ 对筛选出的点集合，进行中心化处理，去掉那么偏远的点
	+ 针对剩余的点，如果剩余的点大于300个(根据实际情况调整)，认为渔漂存在，计算出中心点作为渔漂的中点，计为渔漂初始点
	+ loop: 有超时设置
		+ 继续抓取图片
		+ 重复以上步骤，获取渔漂的点
		+ 如果渔漂的点存在，并且与初始点的 x^2+y^2 > 1000，鱼上钩
		+ 鱼上钩后，点击鱼漂收竿
		+ 若超时，收竿
